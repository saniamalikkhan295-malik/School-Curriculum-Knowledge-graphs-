# School Curriculum Knowledge Graphs

Knowledge graph of Pakistani school curriculum for automated question generation.  
**FYP Project | MS Software Engineering @KUST**

## About
This project builds an ontology-based knowledge graph using Pakistani school textbooks from Grade 1-12. 
Goal: Enable automated question generation for teachers and students.

## Tech Stack
- **Python** - Data processing
- **RDF/OWL** - Ontology modeling  
- **SPARQL** - Querying knowledge graph
- **Neo4j** - Graph database

## Current Status
🚧 Work in Progress - Initial setup

## Future Goals
1. Extract concepts, topics, relationships from textbooks
2. Build RDF triples for curriculum
3. Query engine for MCQs + Short questions
4. Teacher dashboard
